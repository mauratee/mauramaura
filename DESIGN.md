# Creative Portfolio & Shop - Design Document

## 1. Inspiration Analysis

### Reference Sites

| Site | Aesthetic | Key Traits |
|------|-----------|------------|
| **Reference A** | Gallery/boutique | Minimalist grid, neutral palette (off-white, near-black, beige), geometric sans-serif, generous whitespace, high-contrast product imagery |
| **Reference B** | Retro/nostalgic | Windows 95/98 aesthetic, playful and unconventional, beveled borders, system fonts |
| **Reference C** | Clean portfolio | Single-column card layout, content-first, simple navigation, horizontal dividers, professional and approachable |

### Reference A - Gallery Boutique Site

- **Layout:** Minimalist 12-column grid, sticky header, responsive breakpoints
- **Colors:** Off-white (#fcfcfc), near-black (#13131a), warm beige (#eceae7)
- **Typography:** Geometric sans-serif headings (bold, uppercase) + humanist sans body (400-700 weight)
- **Navigation:** Hamburger on mobile, progressive disclosure, content-focused
- **Images:** High-contrast, centered compositions, lazy-loaded
- **Mood:** Curated, refined, upscale boutique experience

### Reference B - Retro Portfolio Site

- **Framework:** Retro CSS framework (Windows 95/98 recreation)
- **Elements:** Gray beveled buttons, inset/raised surfaces, dialog-box styling
- **Typography:** Classic system fonts
- **Mood:** Intentional nostalgia, playful, unconventional

### Reference C - Clean Developer Portfolio

- **Layout:** Vertical single-column, card-based project entries
- **Colors:** Neutral background, dark text, accent via project images
- **Typography:** Clear hierarchy through size/weight variation
- **Navigation:** Minimal - home link, back-to-top button
- **Spacing:** Generous gaps, horizontal rules as dividers
- **Mood:** Professional, approachable, content-focused

## 2. Design Direction

### Approach: "Gallery Minimalist"

A sophisticated, content-first aesthetic that lets artwork take center stage while maintaining a clean, modern e-commerce experience.

**Borrowed from Reference A (Gallery Boutique):**
- Neutral, sophisticated color palette
- Gallery-like whitespace letting artwork breathe
- Grid layout for shop/portfolio sections
- High-quality image presentation as hero elements

**Borrowed from Reference C (Clean Portfolio):**
- Clean, scannable structure
- Card-based artwork/product entries
- Simple top navigation
- Content hierarchy through typography weight

**Borrowed from Reference B (Retro Portfolio):**
- Subtle personality in micro-interactions
- Distinctive character through small details
- Approachable, not sterile

## 7. Accessibility Considerations

- Color contrast: All text meets WCAG AA (4.5:1 minimum)
- Focus states: Visible outline on all interactive elements
- Alt text: Required for all artwork and product images
- Keyboard navigation: Full site navigable without mouse
- Screen reader: Semantic HTML, ARIA labels where needed
- Reduced motion: Respect `prefers-reduced-motion` preference
